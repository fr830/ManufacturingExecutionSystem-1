Webapp - Spring example application Hello World!

Webapp is Java Spring/JSF/database example application. An optimal Java Web stack template (if you do not use Big Data or Spring).

Target server: WildFly (deployable to any Java EE 7 Web Profile application server)

Technologies:

- Spring Boot

- PrimeFaces as JSF UI library

- Liquidbase for database metadata version tracking

- jOOQ for database queries

- PostgreSQL as production database and H2 as unit test database
